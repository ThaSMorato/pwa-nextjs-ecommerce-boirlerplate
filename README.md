---

## Running App Locally
1. Clone repository
2. Run dev
    ```
    $ yarn dev
    ```
3. Access http://localhost:3000/ 
4. Let's Rock!

## Running Production
1. Run build
    ```
    $ yarn build
    ```
2. Run start
    ```
    $ yarn start
    ```
3. Access http://localhost:3000/
